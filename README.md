The **Aadhaar Inclusion Stress Index (AISI)** is a data-driven framework designed to measure **digital inclusion stress** across Indian states and districts using Aadhaar enrollment data.

This project identifies regions where Aadhaar services face higher inclusion challenges and provides actionable insights for policy planning and intervention.
